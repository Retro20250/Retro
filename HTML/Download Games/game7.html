🎂 Comfy Cakes

You run a cake factory where you match cake orders with the correct shape, color, layers, and toppings using machines. It helps build memory sequencing and hand-eye coordination.

👀 Purble Pairs

A classic memory game where you flip cards to find matching images. It trains focus logical thinking and visual memory.

🎭 Purble Shop

A deduction game like “Mastermind” where you guess the correct combination of features for a hidden face. It develops critical thinking color recognition and deduction skills.
